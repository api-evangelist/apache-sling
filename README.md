# Apache Sling (apache-sling)
Apache Sling is a RESTful web framework built on top of the Java Content Repository (JCR). It maps HTTP requests to content resources and uses scripts or servlets to process requests, forming the foundation of Adobe Experience Manager.

## APIs

### Apache Sling
Sling provides a RESTful resource-based API for content management, a Servlet API for request processing, a Scripting API supporting multiple languages (JSP, HTL, Groovy), and a Resource API for JCR content access.
